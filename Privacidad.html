Política de Privacidad - Lumer

Última actualización: Enero 2026

Lumer se compromete a proteger la privacidad de sus usuarios. Esta política explica qué información recopilamos, cómo la usamos y cómo la protegemos.

1. Información que recopilamos

  • Datos proporcionados por el usuario (correo electrónico, nombre opcional).
  • Datos técnicos del dispositivo (modelo, versión de Android, estado del permiso de cámara y ubicación).
  • Datos necesarios para el funcionamiento del servicio (imágenes, notas de voz, fotografías de entrega, ubicación aproximada).

2. Cómo usamos la información

  • Para permitir la creación de cuentas.
  • Para publicar o aceptar envíos dentro de Lumer.
  • Para mejorar seguridad mediante evidencias de entrega.
  • Para enviar notificaciones relacionadas con tu actividad en la app.

3. Con quién compartimos la información

No vendemos ni compartimos tus datos con terceros, excepto cuando sea necesario para procesar pagos (Stripe) o cumplir con normativas legales.

4. Seguridad

Usamos cifrado, almacenamiento seguro y prácticas modernas de protección de datos para mantener tu información personal a salvo.

5. Permisos de la app

La app puede solicitar:

  • Cámara: Para tomar evidencias de entrega.
  • Micrófono: Para notas de voz.
  • Ubicación: Para verificar coordenadas de entrega.
  • Acceso a archivos: Para cargar imágenes o documentos.

6. Datos de menores

Lumer no está dirigida a menores de 13 años y no recopilamos información personal de niños de forma intencional.

7. Contacto

Si tenés dudas sobre esta política, escribinos a: soporte.lumer.app@gmail.com
